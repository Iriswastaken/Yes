# WebSocket Echo Test

This is an example of a ping WebSocket server.

## Quickstart

1. Clone this repository
```sh
git clone https://github.com/tomphttp/websocket-echo.git
cd websocket-echo
```
2. Install dependencies
```sh
npm install
```
3. Run the server
```sh
node ./app.mjs --port 80 --host 127.0.0.1
```
